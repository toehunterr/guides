Here is how you can join your Soulmask dedicated server using the IP address.

## How to Connect to Your Soulmask Server

1. Access the [Game Host Bros Panel](https://panel.gamehostbros.com/).
2. Click on your Soulmask server.
3. Copy the [server IP address](https://www.youtube.com/watch?v=Ap7DtDF-wYg) from the sidebar.
4. Open Soulmask and click `Online Game > Connect to the server directly`.
5. Enter your IP address, port and click `Confirm`

:::caution
If you've set a password, you must enter it here otherwise the server will kick you back to the main menu.
:::
